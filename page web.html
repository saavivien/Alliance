Introduction

Un site web est un ensemble de
pages web liés par des liens permettant
d'aller d'une page à une autre.
Une page web est la description d'un code
HTML.

Le langage HTML est donc à
la base de la création des sites web.
Les avantages du HTML sont
nombreux:

  • Peu couteux

  • Relativement facile à aborder

  • multi-plate-forme

    •  aller à l'autre page
